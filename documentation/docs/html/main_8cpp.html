Recursive loop to find path to goal and backtrack!

Follows a heuristic algorithm of navigating in the following order from any location. Explores North, East, South and West.

ystart/current location's column.

gygoal location's column.
mazeMaze/Map of the environment.
Returns
Final return is true when goal node is reached, else false.

Function to read the input file!

Reads file from the input location and stores in a binary/numeric map. Several things to be taken note of here:

  1. (#) - obstacle, written as 0 to the map
  2. ( ) - free-space, written as 1 to the map

    mazemap where numeric values corresponding to input file is stored.
